# 2026 미국 월드컵 트래커 — 설계 문서

작성일: 2026-06-17

## 목적
블로그 일상란(`category: life`)에 2026 FIFA 월드컵(미국·캐나다·멕시코) 진행 현황을
한눈에 보는 라이브 대시보드를 만들고, 매일 오전 8시(KST)에 결과를 자동 갱신한다.

## 결정 사항 (사용자 승인)
- 갱신 방식: **완전 자동 발행** — 매일 오전 8시 결과 갱신 후 바로 push→Vercel 배포
- 대진표: **HTML/CSS 비주얼 브래킷** (32강→16강→8강→4강→결승)
- 갱신 시각: **매일 오전 8시(KST)** (미국 현지 경기가 한국 새벽에 끝나므로 전날 결과 반영)

## 구조
단일 라이브 포스트 `src/content/blog/2026-world-cup-tracker.md` (category: life).
페이지 구성(위→아래):
1. 대회 개요 — 개최국, 기간(2026.6.11~7.19), 방식(48개국·12개조·32강 토너먼트), 진행 단계
2. 조편성 & 조별 순위 — A~L 12개 조 카드(승점·득실, 상위 2팀 진출권 강조)
3. 비주얼 토너먼트 브래킷 — 32강~결승 CSS 컬럼 브래킷, 모바일 가로 스크롤
4. 다가오는 경기 — 향후 경기 일정
5. 마지막 업데이트 시각

CSS는 포스트 상단 `<style>` 블록에 `.wc-*` 스코프 클래스로 자체 포함.
데이터(팀명·국기·스코어·승점)만 매일 변하고, HTML 골격은 고정 → 에이전트가 안정적으로 채움.

## 자동 갱신 흐름 (매일 08:00 KST, /schedule 루틴)
1. 웹 검색으로 전날 경기 결과·순위 확인 (FIFA·NBC·ESPN·연합뉴스 등)
2. 마크다운의 순위 카드·브래킷·일정 갱신 + `updatedDate` 갱신
3. `npm run build` 통과 확인
4. git add/commit/push → Vercel 자동 배포
5. 텔레그램으로 "갱신 완료 + 주요 결과 요약" 알림

## 안전장치
- 결과 불확실/상충 시 추정 금지 → "미확정" 표기 (블로그 안전규칙 준수)
- 경기 없는 날은 `updatedDate`만 갱신
- 빌드 실패 시 push 중단하고 텔레그램 경고
