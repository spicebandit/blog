# BASELOAD 디자인 리뉴얼 — C안 (프리미엄 신문형) 스펙

2026-07-02 확정. Claude Design 시안 `BASELOAD 리뉴얼 시안.dc.html`의 **2c 옵션** 기반.

## 확정 사항

- **방향**: 프리미엄 신문형 (FT/이코노미스트 계열) — 세리프 헤드라인, 크림 지면, 이중괘선
- **액센트**: 레드 `#C8102E` **단일** — 기존 카테고리 4색(주황/초록/보라/핑크) 폐지. 카테고리는 라벨 텍스트로만 구분
- **다크모드**: 유지 — 신문형 다크 팔레트 신규 설계 (잉크 지면 반전)
- **구독**: 뉴스레터 시스템 없음 → 상단바·브리핑 박스의 구독 액션은 **RSS(/rss.xml)로 연결**
- **인기글**: 사이드바 "가장 많이 읽은 글" TOP은 **GA4 데이터 자동 주입** (`src/data/popular.json`, 없으면 최신글 폴백)
- **배포**: `redesign-c` 브랜치 → Vercel 프리뷰로 먼저 확인, **최종 확정 후에만 main 반영**

## 레이아웃

### 헤더 (2c)
1. **검은 상단바**: 왼쪽 오늘 날짜(한국어) · 오른쪽 "RSS 구독"
2. **마스트헤드 행**: BASELOAD(세리프 30px) + 카테고리 내비(활성=레드 배경 흰 글씨) + 오른쪽 태그라인 "에너지 · 경제 · AI 심층 리포트"
3. 아래 **3px double** 괘선
4. 모바일: 상단바 유지, 마스트헤드 중앙, 내비 가로 스크롤

### 홈
- **히어로**: 최신(또는 지정) 글 — 카테고리 라벨(레드 캡스) + 세리프 대제목 + 요약 + 메타, **오른쪽에 이미지**(4:3)
- 히어로 아래 2단 서브기사 (괘선 구분)
- **사이드바**(340px, surface 배경): "가장 많이 읽은 글" 랭킹(레드 숫자) + "주간 브리핑" 박스(잉크 배경, RSS 링크)
- 이하 카테고리 섹션은 신문형 카드로 통일

### 글 상세
- 세리프 대제목 + 레드 카테고리 라벨, 본문 세리프 17px/1.75
- 표·인용 등 기존 콘텐츠 스타일을 새 팔레트로 매핑
